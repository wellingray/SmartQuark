# SmartQuark - 智能转存与广告清理助手 🚀

[![Platform](https://img.shields.io/badge/platform-Windows-blue.svg)](#)
[![Python Version](https://img.shields.io/badge/python-3.8+-green.svg)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](#)

**SmartQuark** 是一款专为内容创作者、社群运营者及资源分享达人设计的 Windows 桌面端智能效率工具。它可以从您粘贴的任意微信群公告、文章或对话文案中，自动提取出夸克网盘分享链接，将其一键批量转存至您的个人网盘，自动扫描并彻底剔除其中夹带的第三方牛皮癣广告文件，最后以原样格式输出替换了新分享链接的文案，支持一键复制发布！

---

## 🌟 核心功能

*   🔗 **智能链接提取**：您无需手动整理链接，只需直接将微信群公告、聊天记录或网页整段粘贴进软件，软件会自动通过正则过滤出所有独立的夸克分享链接。
*   📂 **日期自动归档 (年份/月份/日期)**：所有转存的资源会自动保存至网盘内的 `/微信群日更/YYYY/MM/DD/` 目录下（如 `/微信群日更/2026/06/16/`），且如果目录不存在，软件将自动一键建好。
*   📦 **批次自动打包**：为避免每天分享的大量资源散落在日期文件夹下，同一批次转存的资源会自动收纳在以“首个资源名称等”命名的二级文件夹中，整洁有序。
*   🧼 **广告自动清洗**：转存成功后，软件会自动遍历文件夹内容，一旦检测到文件名中包含“微信”、“扫码”、“获取更多”、“加群”等引流敏感词的文件，将自动进行物理删除，并单独在界面右下角展示“广告清理日志”。
*   📝 **完美还原排版**：转换完成后，保留您输入的文案段落、空格、标题等所有非链接文字，仅精准将旧链接替换为您的新转存链接，并提供“一键复制新文案”按钮。
*   🔒 **配置本地化**：您的夸克 Cookie 以及敏感词列表会自动持久化保存在本地 [gui_config.json](file:///D:/Antigravity/tools/SmartQuark/gui_config.json) 中，下次打开自动填充，无需反复复制。
*   ⚡ **多线程防死锁**：所有的转存和网络请求全部移交至后台守护线程运行，彻底解决因网络卡顿导致 GUI 窗口“无响应”或假死的问题。

## 🛠️ 安装与部署说明

### 1. 克隆/下载仓库到本地
```bash
git clone https://github.com/您的用户名/SmartQuark.git
cd SmartQuark
```

### 2. 安装 Python 环境
确保您的 Windows 系统上已安装 Python 3.8 或更高版本。

### 3. 安装依赖项
本项目仅需第三方网络库 `requests`，图形库采用 Python 自带的 `tkinter`，轻量且高兼容：
```bash
pip install requests
```

### 4. 运行软件
在控制台或终端运行：
```bash
python smart_quark.py
```

## 🔑 夸克网盘 Cookie 获取方法

由于夸克网盘没有公开的第三方接口，工具需要您的 Cookie 来模拟网页端的转存与分享操作：

1. 用浏览器（Chrome、Edge 等）打开并登录 [夸克网盘网页版](https://pan.quark.cn/)。
2. 登录成功后，按下键盘上的 **F12** 键（或右键选择“检查”）打开开发者工具。
3. 切换到 **网络 (Network)** 选项卡。
4. 刷新网页，在网络请求列表中任意点击一个名为 `info`、`sort` 或 `list` 的请求。
5. 在右侧的 **标头 (Headers)** -> **请求标头 (Request Headers)** 中，找到 **`Cookie`** 字段。
6. 复制其完整的 Value 值，粘贴到 SmartQuark 的 `夸克 Cookie` 输入框中即可。
