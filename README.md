# SmartQuark - 智能转存与广告清理助手 🚀

[![Platform](https://img.shields.io/badge/platform-Windows-blue.svg)](#)
[![Python Version](https://img.shields.io/badge/python-3.8.10-green.svg)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](#)

**SmartQuark** 是一款专为内容创作者、社群运营者及资源分享达人设计的 Windows 桌面端智能效率工具。支持**夸克网盘**与**百度网盘**双平台转存，能自动提取原始文案中的分享链接及密码（提取码），一键批量转存至您的个人网盘中，深度递归扫描并彻底剔除其中夹带的第三方广告、宣传图、引流网页等垃圾文件，生成全新分享链接并完美无缝替换回原排版文案，同时支持将结果一键导出到 Excel 文件！

---

## 🌟 核心功能

*   🔗 **智能链接与提取码识别**：自动通过正则提取出粘贴文案中的夸克网盘链接、百度网盘链接（支持 `s/1...` 与 `init?surl=...` 格式），并精准识别附近的四位提取码或 URL 中的 `?pwd=` 参数。
*   📂 **多网盘深度支持**：
    *   **夸克网盘**：支持极速保存与广告清理。
    *   **百度网盘**：支持全套 Web API 模拟登录、列表查询、防封转存、文件清理和重新创建分享链接。
*   📅 **日期自动归档**：所有转存资源将按结构化归档至网盘中的 `/微信群日更/YYYY/MM/DD/{资源名称}等/` 文件夹下，井然有序。
*   🧼 **广告深度清洗与尺寸启发式规则**：
    *   **深度递归**：递归扫描转存目录下的所有子文件夹。
    *   **图片/网页广告直接删除**：针对常见后缀（`.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.html`, `.url`, `.lnk`），若文件名包含敏感词（如 `微信`、`公众号`、`5分打印`、`必看`、`必读` 等）将直接物理删除。
    *   **尺寸启发式规则**：如果图片大小小于 **150KB** 且文件名中包含任何推广单字（如 `群`、`领`、`扫`、`码` 等），或大小小于 **100KB** 且文件名完全为纯数字、时间戳、减号或下划线（如 `20250727-40652200.jpg`），或为任意网页与快捷方式，将强制作为引流广告删除。
    *   **核心与非核心文档清理**：对于视频、压缩包等大体积资源仅重命名剥离敏感词；对于 `.docx`, `.pdf`, `.txt`, `.epub` 等核心文档，如果文件大小**小于 100KB** 且匹配了敏感词，将被判定为微型引流文档而执行直接物理删除。
*   📝 **完美还原微信文案排版**：转换完成后，微信输出框将保留您输入的段落、空格、文字修饰等格式，仅精准修改链接和提取码。
*   📊 **一键导出到 Excel 文件**：支持将转换成功的 `(资源名称, 资源地址)` 数据一键导出至 Excel 表格中。导出的百度网盘链接已自动重构为纯直连格式（如 `https://pan.baidu.com/s/xxxx?pwd=yyyy`），移除“提取码”等中文字符，方便导入系统或数据库。
*   🔒 **配置自动记忆与合并**：夸克/百度 Cookie 及过滤敏感词列表支持本地 [gui_config.json](file:///D:/Antigravity/tools/SmartQuark/gui_config.json) 持久化保存。Cookie 输入框默认隐藏显示，可按需点击“显示/隐藏”查看；即使升级，新加入的默认广告词也会自动与您的历史配置合并，免去手动维护的麻烦。
*   ⚡ **多线程并发安全**：所有转存和清理操作皆交由后台线程处理，彻底告别窗口“假死”与“无响应”。
*   🛡️ **异常防护与超时机制**：
    *   后台转存流程包含顶层异常防护，任何未预料的错误都不会导致界面卡死，按钮状态自动恢复。
    *   夸克网盘请求默认 30 秒超时；任务轮询设有最大重试上限（120 次）并支持用户手动中断，彻底杜绝线程永久挂起。
    *   百度网盘所有 API 调用失败均记录详细日志（`logging.error`），便于排查 Cookie 过期、接口变更、风控等问题。

## 🛠️ 安装与部署说明

### 1. 克隆/下载仓库到本地
```bash
git clone https://github.com/您的用户名/SmartQuark.git
cd SmartQuark
```

### 2. 运行环境
确保 Windows 系统已安装 Python 3.8.10。

### 3. 安装依赖项
本项目包含网络和 Excel 处理库：
```bash
pip install -r requirements.txt
```

### 4. 运行软件
在控制台或终端运行：
```bash
python smart_quark.py
```

## 🔑 Cookie 获取方法

由于网盘无公开的第三方 API，工具需要您的 Cookie 来模拟网页端的转存与分享操作：

### 夸克网盘 Cookie 获取：
1. 浏览器打开并登录 [夸克网盘网页版](https://pan.quark.cn/)。
2. 登录成功后，按下 **F12** 键打开开发者工具，切换到 **网络 (Network)** 选项卡。
3. 刷新网页，在网络请求列表中任意点击一个名为 `info`、`sort` 或 `list` 的请求。
4. 在右侧的 **标头 (Headers)** -> **请求标头 (Request Headers)** 中复制完整的 **`Cookie`** 值。

### 百度网盘 Cookie 获取：
1. 浏览器打开并登录 [百度网盘网页版](https://pan.baidu.com/)。
2. 登录成功后，按下 **F12** 打开开发者工具，切换到 **网络 (Network)** 选项卡。
3. 刷新网页，在网络请求列表中点击一个请求（如名为 `list` 或 `main` 接口的请求）。
4. 在其 **请求标头 (Request Headers)** 中找到 **`Cookie`** 复制其完整值，粘贴到工具的 `百度 Cookie` 输入框中即可。

## 🔮 编译独立程序 (免环境运行)

如果您需要在没有 Python 环境的 Windows 电脑上运行此软件，可以使用 PyInstaller 将其编译成独立的 `.exe` 应用程序：

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name SmartQuark smart_quark.py
```
编译完成后，可在 `dist/` 目录下找到打包好的单文件可执行文件 `SmartQuark.exe`。
